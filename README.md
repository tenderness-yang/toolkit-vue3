# DevToolkit - 开发者工具集

一个基于 Vue 3 + Element Plus 的现代化开发者工具网站，包含多种实用工具

## 🚀 功能特性

- 🎨 **4 种主题**：浅色/深色/蓝色/绿色，一键切换
- 🔍 **智能搜索**：快速查找工具，支持名称、描述、标签多维度搜索
- 🏷️ **分类筛选**：8 个工具分类，快速定位所需工具
- ⭐ **收藏功能**：收藏常用工具，快速访问
- ⏰ **最近使用**：自动记录最近使用的工具
- 📊 **工具统计**：显示工具热度、使用次数
- ✨ **完整工具**：24+ 个实用工具全部实现
- 📱 **响应式设计**：完美适配桌面、平板、手机
- 🎯 **Element Plus**：现代化 UI 组件，优秀用户体验

## 🎯 页面结构

### 1. 首页 (Home)
- **Hero 区域**：网站介绍和全局搜索

- **热门工具预览**：最常用的 3 个工具
- **统计信息**：工具数量、用户数等数据

### 2. 工具集页面 (Toolkit)
- **智能搜索**：支持多维度搜索
- **快速筛选**：常用、收藏、免费等快速过滤
- **分类浏览**：8 个工具分类卡片
- **热门工具**：6 个最热门的工具
- **所有工具网格**：24+ 个工具卡片
- **最近使用**：自动记录最近 5 个工具
- **统计面板**：工具使用数据统计

### 3. API 文档页面
- API 接口文档（开发中）
- 开发者集成指南

### 4. 关于页面
- 项目介绍
- 技术栈展示
- 联系方式

## 🚦 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 🎨 主题系统

项目支持 4 种主题：
- **浅色主题**：默认主题，适合白天使用
- **深色主题**：护眼模式，适合夜间使用
- **蓝色主题**：专业蓝色调
- **绿色主题**：清新绿色调

主题偏好会自动保存到本地存储。

## 📱 响应式设计

- **桌面端**：完整功能，多列布局
- **平板端**：自适应布局
- **移动端**：单列布局，触摸友好

## 🔧 技术栈

- **Vue 3** - 前端框架
- **Element Plus** - UI 组件库
- **Vite** - 构建工具
- **JavaScript/TypeScript** - 编程语言
- **CSS 变量** - 主题系统

## 📁 项目结构

```
toolkit-vue3/
├── public/              # 静态资源
│   ├── logo.svg        # 网站 Logo
│   ├── favicon.svg     # SVG 图标
│   └── favicon.ico     # ICO 图标（占位符）
├── src/
│   ├── components/     # 组件
│   │   └── ToolModal.vue  # 工具模态框
│   ├── utils/          # 工具函数
│   │   └── tools.js    # 所有工具的实现
│   ├── App.vue         # 主组件
│   └── main.js         # 应用入口
├── index.html          # HTML 模板
├── vite.config.js      # Vite 配置
└── package.json        # 项目配置
```
